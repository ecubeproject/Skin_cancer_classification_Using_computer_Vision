# Dataset Documentation: ISIC 2018 Skin Lesion Analysis

This project uses the publicly available **ISIC 2018 Skin Lesion Analysis Dataset**.  
Due to licensing restrictions, the dataset is **not redistributed** in this repository.  
Follow the instructions below to acquire and prepare the dataset for use.

---

## 1. Download Instructions

The ISIC 2018 dataset can be downloaded from the official source:

ISIC Archive Dataset:  
https://challenge.isic-archive.com/data/#2018

Required components:

- ISIC2018 Task 3 Training Data  
- ISIC2018 Task 3 Training Ground Truth  
- ISIC2018 Task 3 Validation Data  
- ISIC2018 Task 3 Validation Ground Truth  
- ISIC2018 Task 3 Test Data  
- ISIC2018 Task 3 Test Ground Truth (metadata only)
